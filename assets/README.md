# `assets/`

Small local asset bucket kept in the active `core-cats` repository only for files that are still directly used by the current Core-first production path.

Current notable case:

1. `assets/traits/beam.png`
   - the no-logo beam superrare overlay used by the active canonical rebuild path

Boundary note:

1. this directory is **not** a full mirror of the historical Core Cats PNG part library
2. outside readers should not expect all historical coat/accessory/rare-part source files here
3. broader historical raster source art remains part of the `core-cats-eth` archive
4. active canonical outputs in this repository are centered on:
   - `manifests/`
   - `foxar/src/CoreCatsOnchainData.sol`
   - `foxar/src/CoreCatsMetadataRenderer.sol`
   - published viewer artifacts under `manifests/viewer/` and `web/public/viewer/`

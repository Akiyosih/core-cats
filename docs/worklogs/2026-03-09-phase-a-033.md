# 2026-03-09 Phase A-033

## Summary
- fixed Foxar keystore-broadcast scripts to honor `DEPLOYER_ADDRESS` when no raw private key is supplied
- documented that keystore-based pilot deploys must export the keystore-resolved deployer address

## Why
Keystore-backed `spark script --broadcast` runs were falling back to Foxar's default sender when the scripts called `vm.startBroadcast()` without an explicit signer.
That caused pilot deploys to fail even though the correct keystore and password file were present.

# Critical Chain Rule (Do Not Mix Networks)

## Absolute Rule
- In this project, **Core Blockchain** means the `core-coin` ecosystem.
- **Core Blockchain and CoreDAO are different ecosystems.**
- Never use CoreDAO docs/network assumptions for Core Cats chain decisions.

## Authoritative Source Family
- https://coreblockchain.net/
- https://foxar.dev/intro/
- https://github.com/core-coin
- https://github.com/core-coin/ylem
- https://github.com/bchainhub

## Working Rule
- If chain-level information is uncertain, stop and verify before implementation.
- Keep a link trail in docs/commits for any chain settings used in deployment.

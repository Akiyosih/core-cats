# `art/`

Tracked art outputs and review artifacts for the active Core-first path.

This directory is intentionally biased toward:

1. final generated PNG outputs
2. review previews
3. selected token evidence snapshots

It is **not** intended to be the complete historical source-art workspace.

Boundary note:

1. some helper scripts still refer to historical `art/base/...` or `art/parts/...` source paths
2. those references are part of the archive/migration boundary and should not be read as proof that the full historical raster part library is mirrored here today
3. if full historical source art is needed, use the `core-cats-eth` archive as the reference source
